API Products

Flights API

Search, book, and manage flight reservations across 500+ airlines with real-time availability and pricing.

GET /flights/search POST /flights/book GET /flights/{id}

Stays API

Access 50,000+ hotels, lodges, and vacation rentals across Africa with instant confirmation.

GET /stays/search POST /stays/reserve GET /stays/{id}/rooms

Payments API

Accept payments via M-Pesa, cards, bank transfers, and 40+ African payment methods.

POST /payments/initiate GET /payments/{id}/status POST /refunds
