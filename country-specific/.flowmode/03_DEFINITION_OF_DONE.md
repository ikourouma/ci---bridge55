# Definition of Done — Country-Specific

A change is **DONE** only if all of the following are true:

---

## ✅ Governance

- [ ] Blast Radius declared and justified
- [ ] Tier declared and justified
- [ ] Required reviewers included (per matrix)
- [ ] Change Manager readiness completed if Radius >= 3

---

## ✅ Scope Discipline

- [ ] No drive-by refactors
- [ ] Only intended files changed
- [ ] No unrelated cleanup

---

## ✅ Verification

- [ ] Verification steps listed
- [ ] Verification executed (commands + pages checked)
- [ ] No broken navigation paths

---

## ✅ UX / SaaS Integrity

- [ ] No placeholder UI
- [ ] Explicit states: loading, empty, error
- [ ] No dead-end pages
- [ ] Cards represent real, navigable destinations

---

## ✅ Rollback

- [ ] Technical rollback notes included
- [ ] Human/operational rollback included if Radius >= 3
